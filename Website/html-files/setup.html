Setup Guide

Step-by-step instructions to get your ShiroFPV flight controller ready to fly

Setup Image

Prerequisites

Before you begin, make sure you have the following:

  • ShiroFPV flight controller
  • USB cable (appropriate type for your FC)
  • Computer with Betaflight Configurator installed
  • Latest firmware hex file downloaded from GitHub
  • Drone frame with motors and ESCs installed

Step 1: Install Betaflight Configurator

Download and install Betaflight Configurator from betaflight.com/downloads. Choose the appropriate version for your operating system (Windows, macOS, or Linux).

Step 2: Download Firmware

Visit the GitHub releases page and download the latest firmware hex file for your ShiroFPV flight controller.

Flashing the Firmware

Step 3: Connect to Computer

  1. Connect your ShiroFPV flight controller to your computer using a USB cable
  2. Open Betaflight Configurator
  3. Wait for the computer to recognize the device

Step 4: Enter DFU Mode

  1. In Betaflight Configurator, navigate to the "Firmware Flasher" tab
  2. If the FC doesn't automatically enter DFU mode, hold the boot button (if present) while plugging in USB
  3. Verify that the configurator detects the board in DFU mode

Important: Make sure to select "Full chip erase" when flashing firmware for the first time or when switching between major versions.

Step 5: Flash Firmware

  1. Select "Load Firmware [Local]" and choose the hex file you downloaded
  2. Enable "Full chip erase" (recommended for first flash)
  3. Click "Flash Firmware" and wait for the process to complete
  4. Do not disconnect USB during flashing

Initial Configuration

Step 6: Connect and Configure

  1. After flashing completes, disconnect and reconnect the USB cable
  2. Click "Connect" in Betaflight Configurator
  3. Navigate to the "Setup" tab to verify the board is working correctly
  4. Check that the 3D model responds to FC movement

Step 7: Basic Configuration

  1. Ports Tab: Configure UART ports for your receiver, GPS, or other peripherals
  2. Configuration Tab: Set motor protocol, ESC type, and other basic settings
  3. Receiver Tab: Configure your radio receiver and verify stick inputs
  4. Modes Tab: Set up ARM switch and flight modes
  5. Motors Tab: Test motor direction and order (props off!)

Safety First: Always remove propellers when testing motors and configuring your flight controller!

Final Steps

Step 8: Sensor Calibration

  1. Place the drone on a flat, level surface
  2. Go to the "Setup" tab and click "Calibrate Accelerometer"
  3. Wait for calibration to complete
  4. Test that the level indicator is accurate

Step 9: PID Tuning (Optional)

For initial flights, the default PID values should work well. After some test flights, you can fine-tune the PID values in the "PID Tuning" tab to optimize flight characteristics for your specific setup.

Ready to Fly! Your ShiroFPV flight controller is now configured and ready for test flights. Always perform a safety check before flying and start with low-throttle hover tests in a safe area.

Troubleshooting

  • FC not connecting: Try a different USB cable or port. Some cables are charge-only.
  • DFU mode not working: Look for a boot button on the FC or short the boot pads while connecting USB.
  • Motors not spinning: Check motor protocol settings, ESC connections, and motor direction.
  • Receiver not binding: Verify correct UART configuration and receiver protocol settings.

For additional support, visit the GitHub repository issues page or join the FPV community forums.
